# Distilled SCH Beer App

A simple integration of the Brewery API (http://www.brewerydb.com/).

**Tech Stack**

- PHP 5.6.30
- PHPUnit 5.7.14 
- vanilla javascript 
- bootstrap (css only)

**Starting the Project** 

Running PHP Unit: 

```
phpunit --bootstrap autoload.php tests/ 
```

Adding your Brewery API key:

```
classes/BreweryAPI.php:24
```
